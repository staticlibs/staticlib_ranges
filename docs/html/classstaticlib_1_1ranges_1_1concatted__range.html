Detailed Description

template<typename Range1, typename Range2>
class staticlib::ranges::concatted_range< Range1, Range2 >

Lazy implementation of SinglePassRange for concat (or chain) operation, after the pass all accessed elements of source ranges will be moved from (will retain in "valid but unspecified" state).
