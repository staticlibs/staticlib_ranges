Detailed Description

template<typename I1, typename I2, typename E>
class staticlib::ranges::detail::concatted_iter< I1, I2, E >

Lazy InputIterator implementation for concat (or chain) operation.

Does not support CopyConstructible, CopyAssignable and Swappable. Moves element from source iterators one by one and moves it out from operator* method.
