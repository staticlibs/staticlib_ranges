Detailed Description

template<typename R, typename P, typename D>
class staticlib::ranges::detail::filtered_range< R, P, D >

Lazy implementation of SinglePassRange for filter operation, after the pass all accessed elements of source range will be moved from (will retain in "valid but unspecified" state).

Elements that won't match the Predicate will be applied to specified FunctionObject
