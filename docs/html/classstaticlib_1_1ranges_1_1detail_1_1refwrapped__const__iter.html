Detailed Description

template<typename I, typename E>
class staticlib::ranges::detail::refwrapped_const_iter< I, E >

Lazy InputIterator implementation for std::cref operation.

Do not support CopyConstructible, CopyAssignable and Swappable. Wraps elements from source iterator into std::reference_wrapper, and moves wrappers them out from operator* method.
