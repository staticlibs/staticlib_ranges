Detailed Description

template<typename Range, typename Func>
class staticlib::ranges::detail_transform::transformed_range< Range, Func >

Lazy implementation of SinglePassRange for transform operation, after the pass all accessed elements of source range will be moved from (will retain in "valid but unspecified" state).
