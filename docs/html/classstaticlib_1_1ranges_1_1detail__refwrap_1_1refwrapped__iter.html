Detailed Description

template<typename Iter, typename Elem>
class staticlib::ranges::detail_refwrap::refwrapped_iter< Iter, Elem >

Lazy InputIterator implementation for std::ref operation.

Does not support CopyConstructible, CopyAssignable and Swappable. Wraps elements from source iterator into std::reference_wrapper, and moves wrappers them out from operator* method.
